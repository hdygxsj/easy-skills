# Easy Skills

Local Skill Hub for AI IDEs.

## Usage

Use `easy-skills` CLI commands to manage skills.

## Commands

- `easy-skills list --target qoder` - List packages
- `easy-skills status --ide qoder` - Check installations
- `easy-skills info --name <pkg> --target qoder` - Package details

## Install Easy Skills CLI

```bash
go install github.com/easy-skills/easy-skills@latest
```

## Web Interface

Access the web UI at: http://localhost:27842
