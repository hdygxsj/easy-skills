# Easy Skills for Cursor

Install and use Easy Skills Hub in Cursor.

## Quick Install

Tell Cursor agent to fetch instructions:
```
Fetch and follow instructions from http://localhost:27842/cursor/easy-skills.md
```

## Manual Install

```bash
mkdir -p ~/.cursorrules/easy-skills
# Copy easy-skills skill files to this directory
```

## Available Commands

Once installed, you can use:

- `easy-skills list --target cursor` - List available packages
- `easy-skills status --ide cursor` - Check installed packages

## Web UI

Access the Easy Skills Hub at: http://localhost:27842
