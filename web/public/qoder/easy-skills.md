# Easy Skills for Qoder

Install and use Easy Skills Hub in Qoder.

## Quick Install

Tell Qoder agent:
```
/easy-skills Fetch and follow instructions from http://localhost:27842/qoder/easy-skills.md
```

## Manual Install

```bash
mkdir -p ~/.qoder/skills/easy-skills
# Copy easy-skills skill files to this directory
```

## Available Commands

Once installed, you can use:

- `easy-skills list --target qoder` - List available packages
- `easy-skills status --ide qoder` - Check installed packages

## Web UI

Access the Easy Skills Hub at: http://localhost:27842
